# AverageMedian
組込みマイコン向けの移動平均・移動中央値演算クラスライブラリ

## 特徴
* C++で記述されたクラスライブラリ (組込み系でもC++がしたい！)
* malloc, new, STLなどは使わず、メモリは静的に確保する
* テンプレートにより、float や int などの種々の数値型に対応
* 処理の単純化のため、最初の計算は少し手抜き
